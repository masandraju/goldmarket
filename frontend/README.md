# GoldMarket — Frontend

This is the Next.js 15 (App Router) frontend for GoldMarket.

For full project documentation, setup instructions, and deployment guide see the **[root README](../README.md)**.

## Quick start

```bash
npm install
cp .env.example .env.local
# set NEXT_PUBLIC_API_URL and NEXT_PUBLIC_RAZORPAY_KEY_ID in .env.local
npm run dev
```

Runs at `http://localhost:3000`.
